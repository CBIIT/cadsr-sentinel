Using the caDSR Sentinel Tool > Login Page
Login Page
Access to the caDSR Sentinel Tool requires you to have a Login ID and password to enter the system; the Guest account is not valid for using the Sentinel Tool. This is because the Sentinel Tool is designed to notify you specifically regarding changes to the database. In order to do that, the system must know who configured the alert definition.
If you do not have login credentials for the Sentinel Tool, contact the NCICB Help Desk to request one.
When you access the Sentinel Tool URL (http://cadsrsentinel.nci.nih.gov) the Login page appears, requiring a Login ID and password before continuing.
Enter your Sentinel Tool Login ID and Password and click Login. The Sentinel Tool opens displaying the Alerts Page.
