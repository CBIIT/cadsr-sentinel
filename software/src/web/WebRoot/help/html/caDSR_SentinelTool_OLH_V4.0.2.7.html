Using the caDSR Sentinel Tool > Edit Page
Edit Page
The Edit page is used to view, configure and maintain Alert Definitions.
The Edit Alert Definition page contains four tabs, providing details regarding the currently displayed alert.
Editing an alert involves editing the information contained on these tabs.
Properties Tab
Displays the name, created by, created date, modified date and frequency settings.
Report Details Tab
Displays the report content and recipients.
Criteria Tab
Displays the scope of the data to include in the output.
Monitors Tab
Displays the activities that will trigger a report creation.
The Edit page also contains three “action” buttons. After making changes to an alert, select the appropriate button to perform the desired action.
Save
Stores the Alert Definition in the caDSR. When the Save is complete, the Alerts Page appears.
Run
Opens the Manual Alert Run Page. This allows you to make changes to an alert definition and then run the alert without permanently saving the changes. After the alert is submitted for a manual run, the Edit page re-appears with all temporary edits intact.
Manually running an alert does not save the alert to caDSR. You must select Save to save the changes you make to an Alert Definition.
Clear
Restores the information currently resident in caDSR for the displayed alert when viewing a previously saved alert. This is useful if you made temporary changes to an alert for a manual run and want to revert to the original alert definition.
