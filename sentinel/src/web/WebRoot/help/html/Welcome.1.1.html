Welcome to caDSR Sentinel Tool Online Help
The caDSR Sentinel Tool is a web-based tool that allows users to create and manage Alert Definitions for the caDSR database. Alert Definitions are a set of rules that are periodically evaluated against the caDSR.
If the conditions configured in the rules are met, a notification is e-mailed to the user containing a hyperlink to an alert report. Alert reports detail the changes (activities) that have taken place within the caDSR database.
This online help provides information regarding the navigation and use of the caDSR Sentinel Tool to set up alerts.
If you have comments or suggestions for the online help, or if you have questions regarding using the Sentinel Tool, please direct them to the NCICB Help Desk or use one of the Forums on the NCICB GForge site under the caDSR Sentinel Tool project.
