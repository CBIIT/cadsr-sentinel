Using the caDSR Sentinel Tool > Create caDSR Alert Definitions
Create caDSR Alert Definitions
The Create caDSR Alert Definition page is an abbreviated interface to quickly create new Alert Definitions.
Enter an alert name and select the desired Alert Definition template. You have the following choices:
Monitor all changes to my Context(s) with the exception of TEST and TRAINING
This creates a basic alert that monitors activity within any context(s) to which you are associated. The default settings for this alert are as follows:
•
Status is Active, with no end date.
•
Frequency is Daily.
•
Include Properties Section is On.
•
Send Reports is set to only sending a report when there is activity.
•
Recipient is the creator only (you).
•
Static Comment is blank.
•
Forms/Templates, Classification Schemes and Classification Scheme Items are set to All.
•
Monitors are set to Any Change for all properties.
See Edit Page and the subsequent topics on the edit page tabs for information about and instructions for how to change these settings.
Monitor everything created by <your name>
This creates an alert for all changes you make to the caDSR database. The default settings for this type of alert are the same as those listed above.
Start with a blank Alert and allow me to customize it
Selecting this option disables the Save button and requires that you open the Edit Page to configure the settings for the alert.
If you decide not to create a new alert, click Back in the Sentinel Tool interface to return to the Alerts Page. Do not use your browser’s Back button to navigate through the Sentinel Tool as this may cause the tool to function improperly.
