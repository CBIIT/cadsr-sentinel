Using the caDSR Sentinel Tool > Create Using Button
Create Using Button
The Create Using button uses the selected Alert Definition to populate the Edit page for creating a new Alert Definition. The Name is prefixed with "Create Using" and the Created By and Report Recipients are set to the Login ID. All attributes may be edited normally. When complete, select the Save button to record the new Alert Definition or the Back button to cancel the create. Please see Edit Page for a complete description of the page and Alert Definition attributes.
