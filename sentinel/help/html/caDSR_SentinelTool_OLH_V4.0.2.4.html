Login Page
The caDSR Sentinel Tool requires user authentication. By design, users must have a caDSR account to view, create, edit and run Alert Definitions. At NCICB, the Guest account is not valid for the Sentinel Tool. To obtain a user account for the Sentinel Tool, please contact the Alert or System Administrator for the URL on which it was installed. At NCICB, the URL is http://cadsrsentinel.nci.nih.gov.
Enter your caDSR Login ID and Password. The Alerts List Page opens.
