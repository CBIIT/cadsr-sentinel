Using the caDSR Sentinel Tool > Create Using Button
Create Using Button
The Create Using button uses the attributes of the selected Alert Definition to populate the Edit page for creating a new Alert Definition.
When the Edit Page appears, the Name field is prefixed with Create Using and the Created By and Report Recipients fields are set to your Login ID. All attributes may be edited normally.
When complete, select Save to record the new Alert Definition. You may also use the Back button to cancel the alert creation.
