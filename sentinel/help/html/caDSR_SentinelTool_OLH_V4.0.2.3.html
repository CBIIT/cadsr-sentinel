Using the caDSR Sentinel Tool > Alerts Page
Alerts Page
The main page within the Sentinel Tool is the caDSR Alerts page listing all of your alerts. When you first log into the Sentinel Tool, a Welcome message appears in lieu of an alerts list, because you have not yet created any alerts.
Once you have created one or more alerts, those alerts will appear instead of the Welcome page.
All features of the tool are accessible from the Alerts page. From this page, you can create a new alert, edit one of your alerts or view all configured alerts in the database.
The buttons on the Alerts List page provide access to the available actions. These buttons are enabled and disabled as alerts are selected from the list, depending on what actions can be performed on the selected item(s).
To select or deselect an alert, click the checkbox to the left of its name. You may also use the checkbox at the top of the list to select or deselect all alerts in the list.
Each of the available actions is described briefly below. Notice that some actions are only available when one alert is selected as opposed to when multiple alerts are selected.
Only the alert creator can edit, delete, or run their alerts. If you are not the creator of the selected alert, these buttons are disabled.
Create
Allows you to Create caDSR Alert Definitions, opening a form which contains the fields and buttons necessary for creating a new Alert Definition.
Edit
Opens the Edit Page, which allows you to make changes to the selected alert. This button is only enabled when one alert is selected from the list.
Create Using
Opens the Edit Page, which allows you to create a new alert by starting with and making changes to the characteristics of the selected alert. This button is only enabled when one alert is selected from the list. See Create Using Button for a further details on using this feature.
Delete
Allows you to delete the selected alert(s). This button is enabled when one or more alerts is selected from the list. When you select to delete an alert, a confirmation message box appears to avoid accidental deletion. Once confirmed, the selected alerts are permanently deleted from the database.
It is not necessary to delete an alert in order to exclude it from the Auto Run Process. See Edit Page for details on how to change the eligibility of an alert for Auto Run.
Run
Opens the Manual Alert Run Page, which allows you to configure specific run options and then manually run the selected alert against the database. This button is only enabled when one alert is selected from the list.
Show All / Show Private
This button toggles between Show All and Show Private and allows you to select to view all configured alerts or only those that you have created. Show All shows all alerts defined in the database. Show Private shows only the alerts that you have created.
Help
The Help button appears on every page and opens this online help file in a separate browser window.
Logout
The Logout button appears on every page, and when selected terminates the current Sentinel Tool session, returning you to the Login Page.
