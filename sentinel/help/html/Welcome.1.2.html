Accessing the Sentinel Tool
Access to the caDSR Sentinel Tool requires you to have a Login ID and password to enter the system. If you do not have login credentials for the Sentinel Tool, contact the NCICB Help Desk to request one.
To open the caDSR Sentinel Tool, enter the URL, https://cadsrsentinel.nci.nih.gov, into your browser’s address bar. The Login page appears, requesting your Login ID and password information.
You may also access the Sentinel tool through the CDE Curation Tool Options menu.
Use of the caDSR Sentinel Tool has only been certified with Internet Explorer Version 6.0 or higher.
Important:
Please do not use the browser “Back” button within the Sentinel Tool pages. Doing so may cause the tool to function improperly. Each page provides the necessary links and buttons for navigation.
