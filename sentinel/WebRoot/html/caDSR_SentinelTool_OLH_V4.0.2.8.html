Using the caDSR Sentinel Tool > Edit Page
Edit Page
The Edit page is used to view and maintain the Alert Definition attributes.
Select the appropriate action button after completing any changes.
Save
Stores the Alert Definition in the caDSR. Upon successful completion, the List page appears in the browser window.
Clear
Restores the definition from the caDSR when viewing an existing Alert Definition.
Run
Opens the Manual Run Submission page. Any changes made to the Alert Definition are not saved to the caDSR, however the changes are used by the Manual Run. Upon submission from the Run page, the Edit page appears with all unsaved edits intact. See the Run Page for more information.
The Alert Definition attribtues are displayed on four (4) tabs.
Properties
Displays the name, created by, created date, modified date and frequency settings.
Report Details
Displays the report content and recipients.
Criteria
Displays the scope of the data to include in the output.
Monitors
Displays the activities that will trigger a report creation.
The remainder of this section describes the data interdependencies and their effects on the Report output.
