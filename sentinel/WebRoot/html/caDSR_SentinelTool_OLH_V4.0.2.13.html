Using the caDSR Sentinel Tool > Run Page
Run Page
The Manual Alert Run page provides input for a report date range and a selection to use either the creator as the sole recipient or the defined recipient list. The request can then be submitted to the server for the single Alert Definition by selecting the Run button from the List and Edit pages.
The process, called a Manual Run, executes on the web server which hosts the Sentinel Tool. A confirmation message appears after submitting the request. Manual Runs always send an e-mail, the "Send a report from â€¦" specified on the Report Details Tab is ignored. Although the confirmation message appears immediately, the time to create and send the Alert Report depends on many factors including the number of manual runs being processed and the volume of output generated.
A Manual Run executes the same logic and code as an Auto Run with minor exceptions. First, the Manual Run only processes one Alert Definition where the Auto Run may process many. Second, a report is always sent by a Manual Run. When started from the Edit page, an Alert can be run without saving changes to the Definition.
After submitting a Manual Run there is no means to monitor its progress or determine when the Report e-mail will be received.
