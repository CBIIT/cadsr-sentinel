Using the caDSR Sentinel Tool > Create Page
Create Page
The Create page is an abbreviated interface to quickly create new Alert Definitions.
Enter an Alert Name and select the desired Alert Definition template. Attributes not available on this page are set to the following values. Refer to Edit Page for a complete description of each.
•
Status is "Active, with no end date."
•
Frequency is "Run Daily."
•
Include Properties Section is "On."
•
Send Reports is "Only when activity."
•
Recipient is the creator only.
•
Static Comment is "<blank>"
•
Forms/Templates, Classification Schemes and Classification Scheme Items are "All."
•
Monitors are "Any Change" for all properties.
